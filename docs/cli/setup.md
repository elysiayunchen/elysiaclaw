# `elysiaclaw setup`

Initialize `~/.elysiaclaw/elysiaclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
elysiaclaw setup
elysiaclaw setup --workspace ~/.elysiaclaw/workspace
```

To run the wizard via setup:

```bash
elysiaclaw setup --wizard
```
